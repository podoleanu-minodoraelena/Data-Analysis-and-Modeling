# Data Analysis and Machine Learning Portfolio

Acest repository conține o colecție de proiecte de analiză predictivă și modelare statistică realizate în Python. Proiectele demonstrează capacitatea de a prelucra date complexe, de a aplica algoritmi de Machine Learning și de a interpreta rezultatele pentru a extrage valoare de business.

## 📊 Proiecte Principale:

1. **Electricity Cost Prediction (Regression)**
   - **Obiectiv:** Predicția costului energiei electrice pe baza unor indicatori tehnici.
   - **Tehnici:** Regresie liniară multiplă, analiza corelației și vizualizarea datelor.
   - **Relevanță:** Analiză de date aplicată în domeniul energetic.

2. **Water Potability Analysis (Ensemble Learning)**
   - **Obiectiv:** Clasificarea calității apei (potabilă/nepotabilă) folosind parametri chimici.
   - **Tehnici:** Random Forest, AdaBoost și Decision Trees.
   - **Rezultat:** Compararea modelelor de tip ensemble pentru obținerea celei mai bune precizii.

3. **Chicago House Price Prediction (Regularization)**
   - **Obiectiv:** Estimarea prețurilor imobiliare și evitarea overfitting-ului.
   - **Tehnici:** Regresie Ridge și Lasso, optimizarea hiperparametrilor (Alpha).

4. **Student Placement Prediction (Classification)**
   - **Obiectiv:** Predicția șanselor de angajare a studenților pe baza profilului academic.
   - **Tehnici:** Comparare între Gaussian Naive Bayes și Decision Tree folosind matrice de confuzie și curbe ROC.

5. **Wheat Seeds Clustering (Unsupervised Learning)**
   - **Obiectiv:** Identificarea automată a soiurilor de grâu pe baza proprietăților geometrice.
   - **Tehnici:** K-Means Clustering și Gaussian Mixture Models (GMM).

## 🛠️ Tehnologii și Librării utilizate:
- **Limbaj:** Python
- **Analiză de date:** Pandas, NumPy
- **Machine Learning:** Scikit-learn
- **Vizualizare:** Matplotlib, Seaborn
- **Mediu de lucru:** Jupyter Notebook
